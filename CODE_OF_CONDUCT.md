# Codigo de Conducta

Este proyecto adopta un enfoque de colaboracion respetuosa.

## Principios

- Respeto mutuo.
- Critica constructiva.
- Cero tolerancia a ataques personales o acoso.

## Alcance

Aplica a issues, pull requests, discusiones y cualquier canal del proyecto.

## Reportes

Si detectas una situacion inapropiada, abre un issue privado al mantenedor del repositorio.
